## Custom shell theme
### ZSH

`cd ~`

`git clone https://github.com/prabhakaran9397/MyZsh.git`

`cp ~/MyZsh/prabhakaran9397.zsh-theme ~/.oh-my-zsh/themes/`
 
`gedit .zshrc`
 
<p>Change it to ZSH_THEME="prabhakaran9397"</p>
